Construct a collapsible header strip.

Creates and registers a FormaBuffer, constructs a Mapped<bool> whose geometry function produces the header quad, registers it with the Bridge sync loop, and wires the on_press toggle via ctx. Caller adds body elements to layer and calls layer.relate(col.header_id, body_id) for each one; set_visible cascades to all related ids on toggle.

Parameters
windowTarget window for rendering.
layerLayer to register the header element on.
ctxContext to wire the left-click toggle handler.
bridgeBridge owning the sync loop.
cursorLayout cursor. Advanced by row_h on return.
x_minLeft edge in NDC.
x_maxRight edge in NDC.
row_hHeader strip height in NDC units.
initially_openStarting toggle state.
color_closedBackground color when closed.
color_openBackground color when open.
labelOptional GPU image overlaid on the header as a label. When provided, the header uses the multi-texture shader; when null, a plain color-only shader is used.
Returns
Fully constructed Collapsible. cursor_out is below the header strip.
