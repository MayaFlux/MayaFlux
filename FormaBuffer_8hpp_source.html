16 * @brief VKBuffer for 2D screen-space Forma geometry.
17 *
18 * Encodes the invariants that hold for all Forma geometry:
19 * - No depth test, no depth write, no view transform.
20 * - Alpha blending enabled on the RenderProcessor.
21 * - Topology fixed at construction (determines default shaders).
22 *
23 * Provides write_bytes() as the clean write interface for Mapped::sync(),
24 * replacing the raw set_data path on plain VKBuffer.
25 *
26 * Usage:
27 * @code
28 * auto buf = std::make_shared<FormaBuffer>(
29 * initial_capacity_bytes,
30 * Portal::Graphics::PrimitiveTopology::TRIANGLE_STRIP);
31 * buf->setup_processors(ProcessingToken::GRAPHICS_BACKEND);
32 * buf->setup_rendering({ .target_window = window });
33 * @endcode
