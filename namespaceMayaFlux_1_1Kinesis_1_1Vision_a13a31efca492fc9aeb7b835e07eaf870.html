2x box-filter downsample for multi-channel interleaved data.

Each output pixel is the per-channel average of the corresponding 2x2 block. Output dimensions are floor(w/2) x floor(h/2).

Parameters
srcInterleaved float span, size must be w * h * channels.
dstOutput span, size must be >= floor(w/2) * floor(h/2) * channels.
wInput width in pixels.
hInput height in pixels.
channelsChannels per pixel (1 = existing behaviour, 4 = RGBA).
new_wReceives output width.
new_hReceives output height.
