18/**
19 * @class Context
20 * @brief Event wiring between a Layer and a window surface.
21 *
22 * Subscribes to mouse events on a window by constructing Kriya input
23 * coroutines and registering them with a Vruta::EventManager, following
24 * the same pattern as Nexus::Fabric. The EventManager and Window are
25 * injected; Context holds no engine-level singletons.
26 *
27 * Callbacks fire on the scheduler tick. They must not touch GPU resources
28 * directly. State intended for the graphics tick should be written through
29 * shared_ptr captures and read on the next graphics cycle.
30 *
31 * All registered events are cancelled on destruction via
32 * EventManager::cancel_event, keyed by names scoped to this Context.
33 *
34 * Window dimensions are read from window->get_state() at event time,
35 * so resize is handled automatically.
36 */
37class MAYAFLUX_API Context {
38public:
