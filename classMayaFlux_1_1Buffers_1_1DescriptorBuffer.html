Detailed Description

Specialized buffer for shader parameter bindings from nodes.

Binds node outputs to shader uniforms and storage buffers. Designed for parameterizing shaders with live data: time, frequencies, control values, arrays, matrices, etc.

Philosophy:

  • Shaders are controlled by DATA, not hardcoded values
  • Any node can drive any shader parameter
  • Cross-domain flow: audio nodes → visual shader parameters

Usage: ShaderProcessorConfig config; config.shader_path = "parametric.comp"; config.bindings = { {"time", {.binding = 0, .type = vk::DescriptorType::eUniformBuffer}}, {"spectrum", {.binding = 1, .type = vk::DescriptorType::eStorageBuffer}} };

auto buffer = std::make_shared<DescriptorBuffer>(config); buffer->bind_scalar("time", time_node, "time"); buffer->bind_vector("spectrum", fft_node, "spectrum");

auto compute = std::make_shared<ComputeProcessor>(config); buffer->add_processor(compute) | Graphics;
