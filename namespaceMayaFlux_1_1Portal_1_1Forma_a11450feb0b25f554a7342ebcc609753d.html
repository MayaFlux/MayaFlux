Construct a collapsible header followed by N value rows under it.

The header label may include its own dynamic data if the caller wants it visible while collapsed; once expanded, all rows in values become visible via the relation cascade.

Each value row is a separate Element relate'd to the header. Visibility tracks the header's open state.

Parameters
header_labelStatic text composited into the header's label image.
valuesSpecs for each body row. Order is preserved top-to-bottom.
initially_openDefault false so deep trees stay collapsed at construction.
