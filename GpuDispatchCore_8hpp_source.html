25/**
26 * @class GpuDispatchCore
27 * @brief Non-template base that owns all type-independent GPU dispatch logic.
28 *
29 * Separates resource management, buffer staging, and dispatch orchestration
30 * from the type-parameterised boundary in GpuExecutionContext. All virtual
31 * override points that do not reference InputType/OutputType live here so
32 * that implementations can be placed in a .cpp file.
