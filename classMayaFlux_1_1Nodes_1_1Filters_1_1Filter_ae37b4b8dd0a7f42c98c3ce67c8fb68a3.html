Updates filter-specific context object.

Parameters
valueThe current output sample value

Updates FilterContext object that contains information about the filter's current state, including the current sample value, input/output history buffers, and coefficients. This context is passed to callbacks and conditions to provide them with the information they need to execute properly.

The FilterContext allows callbacks to access filter-specific information beyond just the current sample value, enabling more sophisticated monitoring and analysis of filter behavior.
