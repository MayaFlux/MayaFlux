Dispatch to GPU, selected by ExecutionContext mode.

Called by ComputeOperation::apply_operation_internal when a GPU backend is attached and ready. CHAINED mode routes to dispatch_gpu_chained for multi-pass execution; all other modes use dispatch_gpu.

Parameters
inputInput data to process.
ctxExecution context. CHAINED mode requires pass_count and pc_updater keys in execution_metadata.
