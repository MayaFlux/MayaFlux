Detailed Description

Result of make_collapsible().

Owns all Forma constructs produced for one collapsible header. The header element is registered in the layer; any body elements the caller adds via layer.relate(header_id, body_id) will be shown and hidden automatically when the open state changes. cursor_out is positioned at the bottom of the header strip, ready for the first body element.
