13enum class PolynomialMode : uint8_t {

105 * @class Polynomial
106 * @brief Generator that produces values based on polynomial functions
107 *
108 * The Polynomial generator creates a signal based on mathematical functions
109 * that can operate in different modes:
110 * - Direct mode: evaluates f(x) where x is the current phase or input
111 * - Recursive mode: evaluates using current and previous outputs
112 * - Feedforward mode: evaluates using current and previous inputs
113 *
114 * This flexible approach allows implementing various types of polynomial
115 * equations, difference equations, and recurrence relations.
