std::string_view  text,
const PressParams &  params = {},
const std::shared_ptr< Buffers::VKBuffer > &  staging = nullptr

Re-composite a UTF-8 string into an existing GPU texture.

Uploads in-place when content fits within the existing image dimensions. Reallocates and updates target when content height exceeds image height.

Parameters
targetVKImage to update. May be replaced on reallocation.
textUTF-8 string to composite.
paramsrender_bounds, atlas, color.
stagingOptional staging buffer for GPU upload.
Returns
True on success.
