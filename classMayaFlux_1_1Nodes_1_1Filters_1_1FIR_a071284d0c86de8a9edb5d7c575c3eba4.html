double MayaFlux::Nodes::Filters::FIR::process_sample ( double  input = 0.)
overridevirtual

Processes a single sample through the FIR filter.

Parameters
inputThe input sample
Returns
The filtered output sample

Implements the FIR filtering algorithm, computing the weighted sum of the current input and previous inputs according to the filter coefficients. This is the core processing method called for each sample.

Implements MayaFlux::Nodes::Filters::Filter.
