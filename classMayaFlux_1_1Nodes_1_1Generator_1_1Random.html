Detailed Description

Node wrapper for Kinesis::Stochastic - signal-rate stochastic generation.

Provides continuous stochastic signal generation integrated with the node graph system. This is a thin adapter that connects the core Kinesis::Stochastic infrastructure to the processing graph, adding amplitude scaling, callbacks, and GPU context support.

For direct mathematical usage outside the node system, use Kinesis::Stochastic::Stochastic directly.
