POINT_LIST scatter plot for paired (X, Y[, Z]) series.

Pairs the i-th SPATIAL_X series with the i-th SPATIAL_Y series. If there are more Y series than X series, the last X series is reused. Each pair produces one run of PointVertex in the output bytes.

SPATIAL_Z series map to Z position, enabling a true 3D scatter cloud. A COLOR series modulates per-sample point color: samples are mapped through y_range.normalise() and blended between white and the series base color from palette.

Per-series color is resolved from palette cyclically.

Parameters
x_rangeDomain for X.
y_rangeDomain for Y.
z_rangeDomain for Z.
palettePer-series colors, cycled. {1,1,1} if empty.
point_sizePoint size in pixels.
