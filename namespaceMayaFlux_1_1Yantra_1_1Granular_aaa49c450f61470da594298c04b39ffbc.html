Offline granular workflow: segment, attribute, sort, reconstruct.

Runs the full segment → attribute → sort pipeline on container, then stitches the sorted grains per channel into a SoundFileContainer ready for the standard ContiguousAccessProcessor playback path.

Parameters
containerSource signal data.
grain_sizeSamples per grain.

qualifierScalar to extract from the analysis result. Empty uses type default.

Returns
Populated SoundFileContainer containing reconstructed audio in sorted grain order.
