auto MayaFlux::Kakshya::EigenAccess::view ( ) const -> std::optional<Eigen::Map<const EigenType>>

Get zero-copy Eigen::Map view of data.

Template Parameters
EigenTypeTarget Eigen type (VectorXd, VectorXf, VectorXcd, VectorXcf)
Returns
Optional Eigen::Map pointing to variant's internal memory

Returns std::nullopt if:

  • Type mismatch (e.g., vector<float> → VectorXd requires conversion)
  • Data is empty

Supported zero-copy views:

  • vector<double> → Map<VectorXd>
  • vector<float> → Map<VectorXf>
  • vector<complex<double>> → Map<VectorXcd>
  • vector<complex<float>> → Map<VectorXcf>
