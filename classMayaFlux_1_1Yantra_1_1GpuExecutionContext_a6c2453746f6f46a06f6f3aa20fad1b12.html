Read back GPU results and reconstruct output Datum.

Reads the first OUTPUT or INPUT_OUTPUT buffer back into per-channel doubles when m_staging_floats is non-empty. When m_staging_floats is empty (all inputs pre-staged via set_binding_data), falls back to largest_binding_data_element_count() to determine the readback size. Additional OUTPUT bindings with explicit size overrides are written to result.metadata keyed as "gpu_output_N".
