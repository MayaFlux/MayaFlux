TRIANGLE_LIST bar chart for FREQUENCY or SPATIAL_Y series.

Each FREQUENCY series produces one set of bars tiled uniformly across [x_range.min, x_range.max]. Bar height maps sample value through y_range.

Multiple FREQUENCY series are currently rendered on top of each other in the same X range. Side-by-side or stacked layout is not yet implemented; no three-caller evidence exists for those layouts.

Per-series color is resolved from palette cyclically.

Parameters
x_rangeDomain for X (bar positions).
y_rangeDomain for Y (bar heights).
palettePer-series colors, cycled. {1,1,1} if empty.
